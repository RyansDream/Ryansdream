웹앱 만들기

웹앱을 만들기 위해서는, html 기반에서 움직이는 동적언어인
자바스크립트를 CSS로 꾸며서 만든 웹을, 패키징을 통해서 만드는 것이다

HTML

HTML로는 많은 기능들을 할 수 있는데, 예를 들면 버튼을 삽입할 수 있다.



자바스크립트

자바스크립트는 이 html기반으로 만든 버튼을 눌렀을 때 이벤트가 발생하도록 만들어준다.
코딩 방법은 의식의 흐름대로 가면 된다. 버튼을 만들고, 안에 글귀넣고, 눌렀을때
'무엇을' '어떻게' 할 것인가를 쓰자


조건문

조건문 if을 쓰면 위 day/night 버튼을 한 개의 버튼으로 합칠 수 있다.

if라는 조건문을 써서, 버튼의 값이 night일 때 누르면 야간모드로 바뀌면서 값이 day로 바뀌도록 해놓는다. 그리고 if 조건문에 버튼의 값이 night이 아닐 때에는
주간모드의 코딩을 실행하도록 해놓았기 때문에, day라는 값으로 바뀐 버튼을 누르면
주간모드로 바뀌게 된다
if 조건문은 if(true 조건){true일 때 실행}else{false일때 실행}으로 작성된다,

반복문

반복문을 쓰려면 배열의 개념이 필요하다. 우선 반복문의 함수 양식은
while(반복조건){반복조건 만족할 때 표시} 이다
배열의 개념이 필요한 이유는, 반복할 대상을 배열로 정해놔야 반복을 할 수 있기 때문
배열로 변수를 정해놓고. 그 변수를 반복문을 써서 본문 전체의 특정 문구들을
간편하게 수정할 수 가 있다. 예를 들면 버튼 누를 시 hyperlink가 된 문구들의 색을 바꾸는 것.
