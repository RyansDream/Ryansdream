웹앱 만들기

웹앱을 만들기 위해서는, html 기반에서 움직이는 동적언어인
자바스크립트를 CSS로 꾸며서 만든 웹을, 패키징을 통해서 만드는 것이다

HTML

HTML로는 많은 기능들을 할 수 있는데, 예를 들면 버튼을 삽입할 수 있다.



자바스크립트

자바스크립트는 이 html기반으로 만든 버튼을 눌렀을 때 이벤트가 발생하도록 만들어준다.
코딩 방법은 의식의 흐름대로 가면 된다. 버튼을 만들고, 안에 글귀넣고, 눌렀을때
'무엇을' '어떻게' 할 것인가를 쓰자


조건문

조건문을 쓰면 위 day/night 버튼을 한 개의 버튼으로 합칠 수 있다.

if라는 조건문을 써서, 버튼의 값이 night일 때 누르면 야간모드로 바뀌면서 값이 day로 바뀌도록 해놓는다. 그리고 if 조건문에 버튼의 값이 night이 아닐 때에는
주간모드의 코딩을 실행하도록 해놓았기 때문에, day라는 값으로 바뀐 버튼을 누르면
주간모드로 바뀌게 된다
